1. Purpose of the program
The program is intended for use in higher educational institutions in order to demonstrate the main concepts of OO programing.
2. Conditions for execution of the program

Также необходим распространяемый компонент Microsoft Visual C++ 2015
3. Выполнение программы 
Выбор пункта меню осуществляется с помощью нажатия клавиш 0-9. Для возврата на предыдущий экран используется клавиша Backspace.
Для переключения языка пользователь должен нажать клавишу Tab в соответствующем меню.
4. Сообщение оператору
При запуске программы по умолчанию выставлен первый язык, указанныый в конфигурационном файле. Изменить его можно на каждом слайде программы.
При попытке запуска фреймов экрана с неправильно настроенными путями в конфигурационном файле будет выведена ошибка.
В программе имеется проверка на ввод из консоли, поэтому если будут введены некорректные данные, то будет выведено сообщение об ошибке.
